## About Me

Computer Science student with a background in product development, currently focused on cybersecurity.

Primary Focus:
- Hack The Box
- Web Application Security
- Security Product Development

Secondary Interests:
- Capture The Flag (CTF)
- Binary Exploitation (Pwn)

I enjoy learning through hands-on labs, challenges, and building things that solve problems I personally encounter.

I actively participate in CTF competitions without AI assistance during live events. Afterward, I heavily leverage AI for review, alternative approaches, and accelerating the learning process.

Occasionally, I build small tools or products driven by curiosity. For example, I created an HTB XP Calculator after noticing that Hack The Box exposes rank levels but not the XP required to reach them. A bit of research into the XP progression model eventually turned into a project.
