# 👋 About Me

Computer Science student with a background in product development, currently focused on cybersecurity.

## Primary Focus

- Hack The Box
- Web Application Security
- Security Product Development

## Secondary Interests

- Capture The Flag (CTF)
- Binary Exploitation (Pwn)

# Featured Projects

###  HTB XP Calculator

A progression calculator built after noticing that Hack The Box exposes rank levels but not the XP required to reach them.

What started as a small investigation into HTB's XP progression eventually turned into a practical tool for tracking rank goals.

### FileTrust

An experiment focused on helping everyday users make safer decisions when exchanging files online through additional trust and validation signals before opening shared files.
